# Cat Shop System

A Java-based application powered by a MySQL database, designed for feline enthusiasts to browse and purchase cats.

## Overview
The system allows users to create accounts, browse a catalog of available cats, and manage purchases. It handles the lifecycle of a cat from "Available" to "Sold" while maintaining secure records of transactions and order history.

## Features
* **User Management:** Register, authenticate, update profile, and delete accounts.
* **Cat Inventory:** Browse, search, add, update, and remove cat listings.
* **Ordering System:** Create orders, generate individual line items for purchases, and view order history.
